5. A person borrows Rs. 5000 for 2 years at 4% p.a. simple interest. He immediately lends it to another person at 6% p.a for 2 years. Find his gain in the transaction per year?

  
  

View answer

  • Ans: (C)

    • Explanation


    Let P, Q and R represent their respective monthly incomes. Then, we have:

    P + Q = (5050 x 2) = 10100 .... (i)

    Q + R = (6250 x 2) = 12500 .... (ii)

    P + R = (5200 x 2) = 10400 .... (iii)

    Adding (i), (ii) and (iii), we get:  2(P + Q + R) = 33000  or   P + Q + R = 16500 .... (iv)

    Subtracting (ii) from (iv), we get P = 4000.

    P's monthly income = Rs. 4000.

View answer

    • Explanation:

10. The average weight of A, B and C is 45 kg. If the average weight of A and B be 40 kg and that of B and C be 43 kg, then the weight of B is:

  
  

View answer

  • Ans: (D)

    • Explanation:


    Let A, B, C represent their respective weights. Then, we have:

    A + B + C = (45 x 3) = 135 .... (i)

    A + B = (40 x 2) = 80 .... (ii)

    B + C = (43 x 2) = 86 ....(iii)

    Adding (ii) and (iii), we get: A + 2B + C = 166 .... (iv)

    Subtracting (i) from (iv), we get : B = 31.

    B's weight = 31 kg.
