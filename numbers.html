Basic Formulas:

  1. Some Basic Formulae:

    1. (a + b)(a - b) = (a2 - b2)

    2. (a + b)2 = (a2 + b2 + 2ab)

    3. (a - b)2 = (a2 + b2 - 2ab)

    4. (a + b + c)2 = a2 + b2 + c2 + 2(ab + bc + ca)

    5. (a3 + b3) = (a + b)(a2 - ab + b2)

    6. (a3 - b3) = (a - b)(a2 + ab + b2)

    7. (a3 + b3 + c3 - 3abc) = (a + b + c)(a2 + b2 + c2 - ab - bc - ac)

    8. When a + b + c = 0, then a3 + b3 + c3 = 3abc.

10. On dividing a number by 56, we get 29 as remainder. On dividing the same number by 8, what will be the remainder ?

  
  

View answer

  • Ans: (D)

    • Explanation:


    Formula: (Divisor*Quotient) + Remainder = Dividend.

    Soln:

    (56*Q)+29 = D -------(1)

    D%8 = R -------------(2)

    From equation(2),

    ((56*Q)+29)%8 = R.

    => Assume Q = 1.

    => (56+29)%8 = R.

    => 85%8 = R

    => 5 = R.
